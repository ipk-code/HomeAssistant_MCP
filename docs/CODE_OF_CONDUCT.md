# Code Of Conduct

## Our Standard

Contributors are expected to be respectful, constructive, and direct.

## Expected Behavior

- Use respectful language.
- Focus feedback on code, behavior, and design decisions.
- Share security concerns responsibly and with enough detail to reproduce them.
- Help keep documentation and tests accurate.

## Unacceptable Behavior

- Harassment or personal attacks
- Publishing secrets or private data
- Abusive or discriminatory language
- Deliberately bypassing project security controls in submitted changes

## Enforcement

Project maintainers may remove or reject contributions that do not follow these expectations.
